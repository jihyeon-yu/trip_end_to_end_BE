## Trip-End-To-End
개인 맞춤 정보를 활용하여 여행 계획을 세우고 후기까지 작성할 수 있는 여행 계획 웹 사이트

## 기술 스택
* SpringBoot + Maven
* MyBatis
* MySQL
* WebSocket
* Git
* 추가
